# Fujiwara
[![Build Status](https://travis-ci.org/hoto17296/fujiwara.svg)](https://travis-ci.org/hoto17296/fujiwara)

あ゛あ゛あ゛あ゛あ゛あ゛あ゛あ゛！゛！゛！゛

## Install
```
$ npm install fujiwara
```

## Usage
``` js
var fujiwara = require('fujiwara');

var text = 'ああああああああ！！！';
console.log( fujiwara(text) ); // => あ゛あ゛あ゛あ゛あ゛あ゛あ゛あ゛！゛！゛！゛
```
